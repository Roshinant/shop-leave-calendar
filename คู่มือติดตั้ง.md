# 📅 คู่มือติดตั้ง — ตารางหยุดงานร้านเรา

แอปนี้ทำให้ทุกคนในร้านลงชื่อ "วันหยุด" ผ่านมือถือได้ ข้อมูลเก็บกลางบน **Google Sheet**
ทุกเครื่องเห็นตรงกัน + จำกัดวันละ **3 คน** + เปิดออฟไลน์ดูได้

ทำ **2 ส่วน** ตามนี้ (เจ้าของร้านทำครั้งเดียว ประมาณ 15 นาที):

- **ส่วน A** — ตั้งฐานข้อมูล Google Sheet (ให้ข้อมูลแชร์กันได้)
- **ส่วน B** — เอาแอปขึ้นเน็ต ให้ทุกคนเปิดจากมือถือ

> ถ้าอยากลองเล่นก่อนแบบไม่แชร์ข้อมูล แค่ดับเบิลคลิก `index.html` เปิดในเบราว์เซอร์ได้เลย
> (จะเป็น "โหมดเครื่องเดียว" — เก็บข้อมูลในเครื่องนั้นเท่านั้น)

---

## ส่วน A · ตั้งฐานข้อมูล Google Sheet

**1.** เปิด https://sheets.google.com → สร้างสเปรดชีตใหม่ 1 อัน ตั้งชื่ออะไรก็ได้ เช่น "วันหยุดพนักงาน"

**2.** ที่แถบเมนูด้านบน เลือก **ส่วนขยาย (Extensions) → Apps Script**
จะเปิดหน้าต่างเขียนโค้ดขึ้นมา

**3.** ลบโค้ดเดิมที่มีทั้งหมดออก แล้ว **คัดลอกเนื้อหาทั้งไฟล์ `Code.gs`** (อยู่ในโฟลเดอร์นี้) ไปวางแทน → กด 💾 บันทึก

**4.** กดปุ่ม **Deploy (ติดตั้งใช้งาน) → New deployment (การทำให้ใช้งานใหม่)**
- ตรงไอคอนเฟือง ⚙️ เลือกชนิดเป็น **Web app**
- **Execute as (ทำงานในนามของ):** เลือก **Me (ฉัน / อีเมลคุณ)**
- **Who has access (ผู้มีสิทธิเข้าถึง):** เลือก **Anyone (ทุกคน)** ← สำคัญมาก
- กด **Deploy**

**5.** ครั้งแรกจะให้ **อนุญาตสิทธิ์ (Authorize)**
- กด Authorize → เลือกบัญชี Google ของคุณ
- ถ้าขึ้นเตือน "Google hasn't verified this app" ให้กด **Advanced (ขั้นสูง) → Go to ... (unsafe)** → **Allow**
- (ปลอดภัย เพราะเป็นสคริปต์ของคุณเอง)

**6.** จะได้ลิงก์ **Web app URL** ลงท้ายด้วย `/exec`
เช่น `https://script.google.com/macros/s/AKfy....../exec`
👉 **คัดลอกลิงก์นี้เก็บไว้** จะใช้ในขั้นต่อไป

> 🔁 ถ้าแก้โค้ดภายหลัง ต้อง Deploy → **Manage deployments** → กดดินสอ ✏️ → Version: **New version** → Deploy (ลิงก์เดิมใช้ได้ต่อ)

## ส่วน B · เอาลิงก์ใส่แอป แล้วอัปขึ้นเน็ต

### B1. ใส่ลิงก์เข้าไปในแอป (ทำครั้งเดียว)

เปิดไฟล์ `index.html` ด้วยโปรแกรมแก้ข้อความ (เช่น TextEdit / Notepad / VS Code)
หาบรรทัดนี้ใกล้ ๆ ต้นส่วน `<script>`:

```js
const DEFAULT_API_URL = "";
```

วางลิงก์ที่ได้จากส่วน A ลงในเครื่องหมายคำพูด:

```js
const DEFAULT_API_URL = "https://script.google.com/macros/s/AKfy....../exec";
```

บันทึกไฟล์ เท่านี้ทุกเครื่องที่เปิดแอปจะเชื่อม Google Sheet อัตโนมัติ

> อีกวิธี (ไม่ต้องแก้ไฟล์): เปิดแอป → กดปุ่ม ⚙️ มุมขวา → วางลิงก์ → บันทึก
> แต่วิธีนี้ต้องทำทีละเครื่อง จึงแนะนำให้ใส่ในไฟล์ตามด้านบนดีกว่า

### B2. อัปขึ้นเน็ต ให้ทุกคนเปิดได้ (เลือกวิธีใดวิธีหนึ่ง)

การจะเปิดในมือถือแบบ "แอปจริง" + ออฟไลน์ได้ ต้องอัปไฟล์ขึ้นเว็บที่เป็น **https** (เปิดจากไฟล์ในเครื่องตรง ๆ จะใช้ได้ไม่ครบ)

#### วิธีที่ 1 — Netlify Drop (ง่ายสุด ลากวางเสร็จใน 1 นาที) ⭐ แนะนำ
1. ไปที่ https://app.netlify.com/drop
2. **ลากทั้งโฟลเดอร์นี้** (ที่มี `index.html`, `sw.js`, `manifest.webmanifest`, `icon.svg`) ไปวางในหน้าเว็บ
3. รอสักครู่ จะได้ลิงก์ เช่น `https://ชื่อสุ่ม.netlify.app` → **นี่คือลิงก์แชร์ให้พนักงาน**
4. (ไม่บังคับ) สมัครบัญชีฟรีเพื่อเก็บลิงก์ไว้ถาวร / ตั้งชื่อลิงก์เอง

> อัปเดตแอปภายหลัง: ลากโฟลเดอร์ไปวางใหม่ทับได้เลย

#### วิธีที่ 2 — GitHub Pages (ฟรีถาวร)
1. สมัคร https://github.com → สร้าง repository ใหม่ (Public)
2. อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้เข้าไป
3. Settings → Pages → Source: `main` / root → Save
4. รอ 1–2 นาที จะได้ลิงก์ `https://ชื่อคุณ.github.io/ชื่อrepo/`

## 📱 วิธีให้พนักงานใช้ในมือถือ

ส่ง **ลิงก์** (จาก B2) ให้ทุกคนทางไลน์ แล้วบอกให้ทำครั้งเดียว:

**iPhone (Safari):** เปิดลิงก์ → กดปุ่ม **แชร์** ⬆️ → **เพิ่มไปยังหน้าจอโฮม (Add to Home Screen)**

**Android (Chrome):** เปิดลิงก์ → กดจุด 3 จุด ⋮ → **เพิ่มไปที่หน้าจอหลัก (Add to Home screen)**

จะได้ไอคอนแอปที่หน้าจอ กดเปิดได้เหมือนแอปทั่วไป 🎉

## 🌐 มีเน็ต / ไม่มีเน็ต ทำงานยังไง

| สถานการณ์ | ทำอะไรได้ |
|---|---|
| **มีเน็ต** 🟢 | ลงชื่อ / ลบ / ดูได้ครบ · ข้อมูลตรงกันทุกเครื่องทันที |
| **ไม่มีเน็ต** 🔴 | เปิดแอปและ **ดูตารางล่าสุด**ที่โหลดไว้ได้ · แต่ **ลง/ลบไม่ได้** |

> ทำไมแก้ไขตอนออฟไลน์ไม่ได้? เพราะกฎ "วันละ 3 คน" ต้องเช็คกับข้อมูลกลางก่อน
> ไม่งั้นถ้า 2 คนลงพร้อมกันตอนไม่มีเน็ต อาจเกิน 3 คนได้ ระบบจึงบังคับให้ต่อเน็ตตอนบันทึก
> แถบสถานะด้านบนของแอปจะบอกเสมอว่าตอนนี้ 🟢 ออนไลน์ หรือ 🔴 ออฟไลน์

## ❓ ปัญหาที่พบบ่อย

- **แถบขึ้น "ออฟไลน์" ทั้งที่มีเน็ต** → ลิงก์ผิด หรือ Deploy ไม่ได้ตั้ง "Anyone".
  เช็คว่าลิงก์ลงท้าย `/exec` และตอน Deploy เลือก Who has access = **Anyone**
- **ลงชื่อแล้วเด้ง "เต็มแล้ว" ทั้งที่ยังไม่ครบ** → กด "โหลดใหม่" บนแถบสถานะ (อาจมีคนลงพร้อมกัน)
- **แก้โค้ดใน Apps Script แล้วไม่อัปเดต** → ต้อง Deploy เป็น **New version** (ดูหมายเหตุท้ายส่วน A)
- **อยากเปลี่ยนจาก 3 คน เป็นเลขอื่น** → แก้ `MAX_PER_DAY` ทั้งใน `index.html` และ `Code.gs` ให้ตรงกัน
